# 印刷時のページ番号表示（CSS @page マージンボックス）

## 目的・背景

印刷物にページ番号を入れたいが、ブラウザの印刷設定「ヘッダーとフッター」をオンにすると
ページ番号以外（日付・ページタイトル・URL）も一緒に印刷されてしまう。
ページURLはすでに `.print-url`（本文先頭）で自前表示しているため、
ブラウザ設定に頼らずページ番号だけをフッターに出したい。

## 対応方針

`lib/site.js` の印刷CSSに CSS Paged Media の `@page` マージンボックスを追加する。

- `@page { @bottom-center { content: counter(page) " / " counter(pages); } }` で
  各ページ下部中央に「現在ページ / 総ページ数」だけを表示する
- ブラウザの「ヘッダーとフッター」設定はオフのまま使う運用
- 対応ブラウザ: Chrome 131+（2024年末〜）。Firefox / Safari はマージンボックス未対応のため
  何も表示されない（害はないのでフォールバックは設けない）

## 影響範囲

- `lib/site.js`（`layout()` 内の印刷CSSのみ）。全ページ共通レイアウトなので一覧・記事とも同じフッターが付く

## テスト方針

- [x] headless Chrome の print-to-pdf で複数ページになる記事を印刷し、
  各ページ下部にページ番号（例: `1 / 3`）だけが表示されることを確認
  （water-cycle B1-normal 記事＝4ページで `1 / 4`〜`4 / 4` の下部中央表示を確認）
- [x] `npm run build` がエラーなく完走することを確認
